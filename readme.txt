Pour executer l'application ouvrez une session terminal/cmd dans le dossier projet et executer
la commande suivante : python3 app.py

Remarque : vous devez au préalable avoir installé CustomTkinter.

Explication du fonctionnement de l'appli:

-Fenetre principale:
    choisissez la methode de chiffremnet que vous voullez avec les boutons radio sur la gauche.
    choisissez l'action à executer (chiffrer/déchiffrer) avec les radio boutons en haut
    Ecrivez votre texte dans le texte box à gauche et appuyez sur le bouton chiffrer (ou dechiffrer, selon l'action que vous avez choisi) une clé par defaut sera utilisée voir en bas comment definier la clé selon la methode.
    Le resultat s'affichera dans le texte box à droite.
